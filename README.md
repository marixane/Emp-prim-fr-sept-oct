# A4 Photo Maker

Plateforme simple pour créer une page A4 avec 3 ou 4 photos et l'exporter en PDF.

## Fonctions actuelles

- Importer 3 ou 4 photos
- Choisir un modèle automatique, 3 photos ou 4 photos
- Ajouter un titre optionnel
- Prévisualiser la page A4
- Exporter la page en PDF

## Installation

```bash
npm install
npm run dev
```

Puis ouvrir l'adresse affichée dans le terminal.

## Build

```bash
npm run build
```
